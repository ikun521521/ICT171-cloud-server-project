Project Overview

This website was created for the ICT171 Cloud Server Project. The purpose of this project is to demonstrate my ability to deploy, configure, and document a cloud-based Infrastructure as a Service server.

The website is hosted on an Amazon EC2 virtual machine running Ubuntu Server and Nginx. It will be used to showcase my technical skills, cloud server setup, documentation, and project development process.

A Bash script will be added to this server to automatically generate a server status report. The report will show information such as disk usage, memory usage, uptime, and the last updated time.

This scripting component will demonstrate automation and server monitoring as part of the ICT171 project requirements.
